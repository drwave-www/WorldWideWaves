toPolygon

val List<Position>.toPolygon: Polygon(source)

Converts a list of positions to a polygon.

Usage:

val polygon = listOf(
    Position(51.5, -0.1),
    Position(51.5, -0.2),
    Position(51.6, -0.2)
).toPolygon
Content copied to clipboard

This extension property provides a concise way to construct polygons from coordinate lists, avoiding verbose Polygon() construction.

Time Complexity: O(n) where n = number of positions Space Complexity: O(n) for the new Polygon instance

Note: The resulting polygon is NOT automatically closed. Call .close() if you need a closed polygon:

val closedPolygon = positions.toPolygon.close()
