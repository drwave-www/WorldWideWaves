IosMapLibreAdapter

class IosMapLibreAdapter(eventId: String) : MapLibreAdapter<UIImage>

iOS MapLibreAdapter implementation.

Bridges Kotlin shared logic to Swift MapLibreViewWrapper via MapWrapperRegistry. Camera commands are stored in registry and executed immediately via callbacks.

Architecture: Kotlin → MapWrapperRegistry → Swift MapLibreViewWrapper → MapLibre SDK
