class SetEventFavorite(favoriteEventsStore: FavoriteEventsStore)(source)

Convenience wrapper to toggle the favorite flag for a given IWWWEvent. Persists the flag through FavoriteEventsStore and mirrors the change back into event.favorite so callers do not have to mutate the model themselves.
