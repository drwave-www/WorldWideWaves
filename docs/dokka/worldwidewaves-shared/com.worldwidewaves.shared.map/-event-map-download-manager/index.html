EventMapDownloadManager

class EventMapDownloadManager(platformMapManager: PlatformMapManager)(source)

EventMap-specific download manager for map resources.

Provides unified download state management and orchestration logic:

  • Availability checking

  • Progress tracking (0-100%)

  • Error handling

  • Auto-download triggers

  • Multi-map state tracking and cache management

Platform-specific download mechanisms (Play Dynamic Features vs ODR) are delegated to PlatformMapManager implementations.

NOTE: This is distinct from viewmodels.MapDownloadCoordinator which serves the ViewModel layer with adapter-based platform integration.
