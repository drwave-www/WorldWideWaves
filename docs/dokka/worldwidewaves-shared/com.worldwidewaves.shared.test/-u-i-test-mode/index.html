UITestMode

object UITestMode(source)

UI Test Mode singleton for managing test environment state.

Purpose

Provides a global flag to detect when the app is running in UI test mode, allowing different behavior for automated testing (e.g., test data seeding, faster animations, mock services).

Usage

// From iOS (SceneDelegate)
enableUITestMode()

// From Android (MainActivity)
enableUITestMode()

// Check in app code
if (UITestMode.isEnabled) {
    // Use test data or mock services
}

Thread Safety

Test Data Seeding

When enabled, automatically calls TestDataSeeder.seedTestData() to populate:

  • Paris France event in "running" state

  • paris_france map marked as downloaded

  • Clear favorites list
