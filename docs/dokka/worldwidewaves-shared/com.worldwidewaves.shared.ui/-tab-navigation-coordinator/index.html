class TabNavigationCoordinator(platformEnabler: PlatformEnabler, screens: List<TabScreen>, val tabBarItem: @Composable (isSelected: Boolean, tabIndex: Int, contentDescription: String?) -> Unit)(source)

Simple tab navigator for Compose.

Takes a list of TabScreens and a Composable factory (tabBarItem) that draws each item (icon/text). It then:

  1. Renders the currently-selected screen in a flexible content area.

  2. Displays a bottom tab-bar that lets the user switch screens.

  3. Persists the selected tab via remember so configuration changes keep the current selection.
